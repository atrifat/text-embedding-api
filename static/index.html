Text Embedding API

A simple Text Embedding API using FastAPI and Hugging Face Transformers.

Use this API to get text embeddings using various supported model from Hugging Face Transformers.

Endpoints

  • API Documentation

Example Request

curl -X POST "http://localhost:7860/v1/embeddings" -H "Content-Type: application/json" -d '{"input": "Hello, world!"}'

Example Response

{"data":[{"object":"embedding","embedding":[-0.06830555200576782, ... ,0.06830555200576782],"index":0}],"model":"text-embedding-3-large","object":"list","usage":{"prompt_tokens":6,"total_tokens":6}}
